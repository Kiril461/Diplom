# Library Website

Сайт-візитка для бібліотеки, що надає інформацію про бібліотеку, каталог книг, новини та форму зворотного зв’язку.

## Вимоги
- Node.js (16+)
- PHP (8.1+)
- MySQL (8.0+)
- Docker (опціонально)
